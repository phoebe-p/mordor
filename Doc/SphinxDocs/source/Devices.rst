Devices
=======

*Mordor* can have any number of devices configured, although only those useful for a particular application will be available in the corresponding device selector. Section :ref:`AddDevices` gives more details on adding new equipement to *Mordor*.

Currently, these are the devices configured for the different types of instruments:

#. **Monochromators:**

    - :ref:`Dummy monochromator <Dummy-monochromator>`
    - :ref:`Acton SP2500i <ActonSP2500i>`

#. **Spectrometers:**

    - :ref:`Dummy spectrometer <Dummy-spectrometer>`
    - :ref:`Ocean Optics HR4000 <HR4000>`

#. **Lock-In amplifiers:**

    - :ref:`Dummy lockin <Dummy-lockin>`
    - :ref:`SR830 Lock-In <SR830>`

#. **Source-meter units (SMU):**

    - :ref:`Dummy SMU <Dummy-smu>`
    - :ref:`Keithley 236 <Keithley236>`
    - :ref:`Keithley 2430 <Keithley2430>`

#. **Multimeters:**

    - :ref:`Keithley 2000 <Keithley2000>`
